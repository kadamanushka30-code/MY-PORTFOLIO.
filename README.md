**MY PORTFOLIO**
**MyPortfolio** is a responsive personal portfolio website that showcases my skills, projects, resume, and contact details in one place. It is designed to present my work professionally to recruiters and visitors.

**Features**
1. Responsive and mobile-friendly design.
2. Projects section to display my work.
3. Skills section to highlight technical abilities.
4. Resume or CV link for quick access.
5. Contact section for easy communication.

**Tech Stack**
1. Visual Studio Code
2. HTML
3. CSS

**Installation**
1. Clone the repository.
2. Open the project folder.
3. Run the project in your browser.

**Bash**
git clone https://github.com/kadamanushka30-code/MyPortfolio.git

**Usage**
Open the website and browse through the portfolio sections to view my projects, skills, and contact information.

**Purpose**
This project helps me build a professional online presence and makes it easier to share my profile with recruiters, mentors, and potential employers.

**Repository Description**
A responsive personal portfolio website built to showcase projects, skills, and resume details in a clean and professional layout.
